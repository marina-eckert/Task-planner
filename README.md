# Task Planner

Task Planner — это удобное приложение для планирования и отслеживания задач, построенное на основе современного стека: **Vite**, **React**, **TypeScript** и **Tailwind CSS**.

---

## Технологии

- **Vite**
- **React + TypeScript**
- **Tailwind CSS**
- **React Router DOM**
- **React Hook Form + Zod**

---

## 📦 Установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/marina-eckert/Task-planner.git
cd vite-project

# 2. Установить зависимости
npm install

# 3. Запустить приложение
npm run dev
```
